## Shortcuts

- `Ctrl + r`: Search history using fzf
- `Ctrl + t`: Search for file using fzf

## Aliases

- `**`: fzf completion

## Tools

- fd: `fd <pattern>` to search for files
- bat: `bat <file>` to view code
- glow: `glow <file>` to view markdown
